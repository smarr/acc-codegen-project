# Code Generation for Register Machines

This repository contains basic starter code for the 
project option 1 of the Advanced Compiler Construction module.

### Main Code

 - codegen: Java examples for code generation and Hello World! programs, as well as the SimpleLanguage grammar

 - codegen/launcher.c: example wrapper to run the code generated by the examples

 - tests: example programs of the SimpleLanguage

### Scripts

The following scripts are intended as illustration and are tested on macOS (ARM64) and Ubuntu (x86-64).

 - `./assembly.sh <filename.asm>` converts the textual assembler into machine code that can be used by launcher.c

 - `./disasm.sh <filename.bin>` disassembles a plain binary file

 - `./run.sh <filename.sl>` runs the parser on a Simple Language program to check its syntax

 - `./run-example-codegen.sh` runs the Hello World example code generator and produces a .bin and .asm file as output

### Useful Commands

 - `./run.sh tests/test.sl` runs the SimpleLanguage parser on the test program

 - `gobjdump -D -b binary -m i386:x86-64 hello-x86_64.bin` disassembles the binary for x86

 - `gobjdump -D -b binary -m aarch64 hello-aarch64.bin` disassembles the binary of ARM64

 - On MacOS it is `gobjdump`, on Linux typically `objdump`
